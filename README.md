# Portfolio Website - Hari Charan Bonam

A modern, responsive portfolio website built with React and Vite showcasing my projects, skills, and professional experience.

## 🚀 Features

- **Modern Design**: Beautiful gradient effects, smooth animations, and a dark theme
- **Responsive**: Fully responsive design that works on all devices
- **Interactive**: Smooth scroll navigation, hover effects, and micro-interactions
- **Fast**: Built with Vite for lightning-fast development and optimized production builds
- **Easy to Edit**: All personal data is stored in a single `portfolioData` object at the top of `App.jsx`

## 🛠️ Technologies Used

- React 18
- Vite
- CSS3 (Custom animations & effects)
- Lucide React (Icons)

## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```

## ✏️ Customization

All your personal information can be easily edited in the `portfolioData` object at the top of `src/App.jsx`:

- **Personal Info**: Name, email, phone, location
- **Social Links**: GitHub, LinkedIn, LeetCode, etc.
- **Skills**: Languages, frameworks, tools
- **Education**: Degrees and institutions
- **Experience**: Work history and achievements
- **Projects**: Your project showcase
- **Certifications**: Professional certifications
- **Achievements**: Awards and recognitions

## 🎨 Color Scheme

The portfolio uses a modern dark theme with vibrant accents:
- Primary: Indigo (#6366f1)
- Secondary: Emerald (#10b981)
- Accent: Amber (#f59e0b)
- Background: Dark (#0a0a0f)

You can customize these colors in `App.css` by modifying the CSS variables in the `:root` selector.
